# Getting and Cleaning Data - Coursera Project

## Modifications
1. Merging of training and test dataset to create one dataset
2. Extracting only mean and std variables
3. Using feature and activity files to name variables
4. Creating a tidy dataset containing only mean for each of the variables

## Identifiers

* Subject - ID of Subject
* Activity - Name of the activity performed by subject at time of measurement
